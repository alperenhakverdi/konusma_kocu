# 🎤 TalkSense AI - Yapay Zeka Destekli Konuşma Koçu

**TalkSense AI**, kullanıcıların konuşma becerilerini geliştirmelerine yardımcı olan interaktif bir platformdur. Gerçek zamanlı ses analizi ve Gemini AI gücüyle, etkili ve kişiselleştirilmiş geri bildirimler sunar.

---

## ✨ Özellikler

- 🎙️ **Gerçek Zamanlı Ses Analizi** – Web Speech API ile anlık ses-metin dönüşümü  
- 🧠 **AI Destekli Değerlendirme** – Gemini AI ile detaylı konuşma analizi  
- 📊 **Görsel Raporlama** – Radar grafiklerle skor görselleştirmesi  
- 🎯 **Kişiselleştirilmiş Öneriler** – Bireysel gelişim için öneriler  
- 📱 **Modern Arayüz** – Responsive tasarım ve animasyonlu kullanıcı deneyimi

---

## 🚀 Nasıl Çalışır?

1. 🎤 **Konuşun** – Mikrofona konuşmanızı kaydedin  
2. 🤖 **Analiz Edin** – Konuşmanız yapay zeka ile analiz edilir  
3. 📈 **Sonuçları Görün** – Skorlar ve gelişim önerileri alın  
4. 🎯 **Kendinizi Geliştirin** – Önerilere göre pratik yapın  

---

## 💡 Kullanım Alanları

- İş görüşmesi ve mülakat pratiği  
- Sunum teknikleri geliştirme  
- Diksiyon ve telaffuz çalışmaları  
- Topluluk önünde konuşma pratiği  

---

## 📊 Değerlendirme Kriterleri

- **Akıcılık (0–100):** Duraksamalar, tekrarlar, dolgu kelimeler  
- **Dilbilgisi (0–100):** Gramer ve cümle yapısı  
- **İçerik Kalitesi (0–100):** Anlam bütünlüğü, derinlik  
- **Ses Tonu (0–100):** Etkileyicilik, profesyonellik  

**Detaylı Geri Bildirim:**  
- Güçlü yönler ve gelişim alanları  
- Duygusal analiz (özgüven, enerji, samimiyet)  
- Pratik öneriler  

---

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** React.js, Tailwind CSS, Chart.js  
- **Backend:** Node.js, Express  
- **AI:** Google Gemini API  
- **Ses İşleme:** Web Speech API  

---

## ⚙️ Kurulum

**Gereksinimler:**
- Node.js (v16+)  
- Google Gemini API anahtarı  

```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm start
````

`.env` dosyasına aşağıdaki satırı ekleyin:

```
GEMINI_API_KEY=your_api_key_here
```
